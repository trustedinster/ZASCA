本地开发环境启动步骤

1. 创建虚拟环境
python -m venv venv

2. 激活虚拟环境
venv\Scripts\activate

3. 安装依赖
pip install -r requirements.txt

4. 数据库迁移
python manage.py migrate

5. 创建管理员账号
python manage.py createsuperuser

6. 启动服务器
python manage.py runserver

访问地址: http://127.0.0.1:8000
后台管理: http://127.0.0.1:8000/admin/
