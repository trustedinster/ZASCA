# Static Images

This directory contains static images for the ZASCA project.

## Required Images

1. **default-avatar.png** - Default user avatar image
   - Size: 200x200 pixels
   - Format: PNG with transparency support
   - Style: Neutral user silhouette

2. **favicon.ico** - Website favicon
   - Size: 32x32 pixels
   - Format: ICO
   - Style: ZASCA logo or initials

## Image Guidelines

- Use SVG or PNG format for better quality
- Optimize images for web (small file size)
- Maintain consistent style across all images
- Consider accessibility (use alt text)
- Use responsive images where appropriate
