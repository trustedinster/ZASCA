# 2c2a 用户手册

欢迎使用 2c2a 云电脑管理平台。本文档将帮助你了解平台的各项功能和使用方法。

<div class="md-alert md-alert-note" markdown="1">
**📖 阅读提示** — 左侧为章节导航，右侧为页面目录，点击即可快速跳转。
</div>

---

## 1. 账户管理

### 1.1 注册账户

1. 在登录页面点击「立即注册」进入注册页面。
2. 完成人机验证（图片验证码 / 滑动验证）。
3. 填写邮箱地址，点击发送验证码，查收邮件并输入 6 位数字验证码。
4. 填写用户名、密码等必要信息，提交注册。

<div class="md-alert md-alert-tip" markdown="1">
**邀请注册**：如果你收到了管理员发送的注册链接，直接访问该链接即可注册，注册成功后将自动加入指定的用户组，获得对应权限。
</div>

### 1.2 登录

在登录页面输入用户名和密码，完成人机验证后点击「登录」。勾选「记住我」可在一定时间内免登录。

### 1.3 忘记密码

1. 在登录页面点击「忘记密码」。
2. 输入注册时使用的邮箱地址，完成人机验证后发送重置验证码。
3. 输入验证码和新密码，完成重置。

### 1.4 个人资料

登录后点击导航栏头像 →「个人资料」，可以：

- 查看和编辑个人信息
- 上传头像
- 修改密码

## 2. 仪表盘

登录后默认进入仪表盘页面，展示以下信息：

- **统计卡片**：云电脑数量、申请数量、工单数量等关键指标
- **图表**：资源使用趋势
- **最近操作**：系统最新动态
- **主机状态**：在线/离线主机概览
- **系统告警**：需要关注的事项

你可以通过「组件配置」自定义仪表盘上显示哪些组件。

## 3. 云电脑

### 3.1 查看我的云电脑

点击导航栏「我拥有的云电脑」，可以查看当前账户下所有云电脑，按产品分组展示。每台云电脑显示：

- 用户名和状态（激活 / 未激活 / 已禁用）
- 所属产品和主机信息
- 磁盘配额使用情况

### 3.2 获取初始密码

新创建的云电脑会生成一个初始密码，采用**阅后即焚**机制：

1. 在云电脑详情页点击「获取密码」。
2. 系统显示初始密码，**此密码仅可查看一次**。
3. 查看后密码将被销毁，无法再次获取。
4. 请务必妥善记录密码。

<div class="md-alert md-alert-warning" markdown="1">
**⚠️ 注意** — 初始密码只能查看一次，请立即复制保存。关闭页面后无法再次查看。
</div>

### 3.3 远程连接（RDP）

1. 在云电脑详情页点击「连接」。
2. 系统将自动下载 `.rdp` 远程桌面文件。
3. 双击打开 `.rdp` 文件，输入云电脑的用户名和密码即可连接。

<div class="md-alert md-alert-tip" markdown="1">
远程连接通过 RD Gateway 安全隧道建立，无需暴露主机公网 IP。
</div>

## 4. 申请开户

### 4.1 提交申请

1. 点击导航栏「我提交的申请」→「创建申请」。
2. 选择需要开通的产品。
3. 填写开户信息：
   - **用户名**：云电脑上的登录用户名
   - **姓名**：真实姓名
   - **邮箱**：联系邮箱
   - **描述**：用途说明
   - **磁盘容量**：如需额外磁盘空间可填写
4. 确认信息无误后提交。

### 4.2 查看申请状态

在「我提交的申请」列表中可以查看每条申请的当前状态：

| 状态 | 说明 |
|------|------|
| 待审核 | 等待管理员审核 |
| 已批准 | 审核通过，等待系统自动创建用户 |
| 处理中 | 系统正在创建云电脑用户 |
| 已完成 | 云电脑已创建成功 |
| 已拒绝 | 申请被驳回 |
| 失败 | 创建过程中出现错误 |

### 4.3 申请详情

点击申请记录可查看详细信息，包括完整的时间线：

- 提交时间
- 审核时间和审核人
- 执行时间和结果

## 5. 邀请链接

部分产品设置为「邀请访问」，需要通过邀请链接才能申请开通。

1. 获取管理员或提供商发送的邀请链接。
2. 访问邀请链接，系统自动解锁对应产品的访问权限。
3. 解锁后即可正常提交开户申请。

<div class="md-alert md-alert-important" markdown="1">
邀请链接可能包含**使用次数限制**和**有效期限制**，请在有效期内尽快使用。
</div>

## 6. 工单系统

工单系统用于提交问题、请求帮助和跟踪处理进度。

### 6.1 创建工单

1. 点击导航栏「工单中心」→「创建工单」。
2. 填写工单信息：
   - **标题**：简要描述问题
   - **描述**：详细说明情况
   - **分类**：选择问题所属类别
   - **优先级**：紧急 / 高 / 中 / 低
   - **关联产品 / 主机**（可选）：便于处理人定位问题
3. 提交工单。

### 6.2 查看工单

- **我的工单**：显示你创建的所有工单。
- **待处理工单**：显示分配给你处理的工单。

工单列表支持按状态、优先级、分类、日期和关键词筛选。

### 6.3 工单状态流转

| 状态 | 说明 |
|------|------|
| 待处理 | 新创建的工单，等待分配 |
| 处理中 | 处理人正在处理 |
| 待反馈 | 处理人需要你提供更多信息 |
| 已解决 | 问题已解决 |
| 已关闭 | 工单关闭 |
| 已驳回 | 工单被驳回 |

### 6.4 工单互动

- **评论**：在工单详情页可以添加评论，与处理人沟通。
- **关闭工单**：问题解决后，你可以手动关闭工单，并填写满意度评分和评价。

### 6.5 工单仪表盘

工单仪表盘展示统计数据：工单总数、各状态数量、超时工单数、优先级分布等。

## 7. 提供商功能

提供商是负责管理主机和产品的角色，拥有以下功能。

### 7.1 主机管理

- 查看分配给自己的主机列表
- 创建新主机（填写名称、地址、端口、连接类型等）
- 编辑主机信息
- 部署主机（生成部署命令，在目标主机上执行）
- 启用 / 禁用主机
- 管理主机组（将多台主机归组管理）

### 7.2 产品管理

- 创建产品（关联到主机，设置可见性、自动审核、磁盘配额等）
- 编辑产品信息
- 管理产品组
- 删除产品

### 7.3 云电脑用户管理

- 查看所有云电脑用户
- 同步远程主机上的管理员
- 设置磁盘配额
- 重置用户密码
- 批量激活 / 停用 / 禁用用户

### 7.4 开户申请审核

- 查看所有开户申请
- 批准或驳回申请（可填写审核备注）
- 执行开户（系统自动在远程主机创建用户）
- 批量批准 / 批量驳回

### 7.5 邀请令牌管理

- 创建邀请令牌（关联产品或产品组，设置使用次数和有效期）
- 启用 / 禁用令牌
- 查看访问授权记录

### 7.6 工单管理

- 查看所有工单
- 添加评论和内部备注（内部备注仅工作人员可见）
- 批量处理工单
- 管理工单分类

## 8. 管理员功能

管理员拥有系统的最高权限，除提供商的所有功能外，还包括：

### 8.1 用户管理

- 查看所有用户列表
- 创建新用户
- 编辑用户信息
- 删除用户
- 切换用户激活状态
- 重置用户密码

### 8.2 用户组管理

- 创建 / 编辑 / 删除用户组
- 管理用户组成员

### 8.3 注册链接管理

- 生成注册邀请链接（关联到用户组）
- 设置链接有效期
- 删除注册链接

### 8.4 提供商分配

- 为主机分配提供商
- 为主机组分配提供商

### 8.5 主机管理（增强）

- 向导式创建主机（分步引导）
- 连接测试（验证主机是否可达）
- 管理所有主机（不限提供商）

### 8.6 系统配置

- **邮件配置**：SMTP 服务器地址、端口、TLS、账号密码
- **验证码配置**：全局和场景化（登录 / 注册 / 邮箱）验证码提供器设置
- **站点配置**：站点名称、注册开关、备案号
- **邮箱限制**：邮箱后缀白名单 / 黑名单

### 8.7 主题配置

- 选择主题风格（Material Design 3 / 新拟态）
- 上传品牌资源（Logo、Favicon、登录背景）
- 自定义颜色方案
- 编辑页面内容（登录页欢迎语、公告、页脚文字等）
- 管理组件布局

### 8.8 审计日志

- 查看系统所有操作的审计记录
- 查看操作详情（操作人、时间、IP、操作内容）

### 8.9 引导系统

- 为主机生成初始配置令牌和配对码
- 查看待配对主机列表
- 触发主机引导
- 吊销待验证主机
- 管理活动会话

## 9. 常见问题

### Q: 忘记密码怎么办？

在登录页面点击「忘记密码」，通过注册邮箱接收验证码重置密码。

### Q: 初始密码查看后还能再看吗？

不能。初始密码采用阅后即焚机制，仅可查看一次。请务必在首次查看时立即保存。

### Q: 为什么看不到某个产品？

部分产品设置为「邀请访问」，需要先通过邀请链接解锁权限才能看到。请联系管理员获取邀请链接。

### Q: 如何远程连接云电脑？

在云电脑详情页点击「连接」，下载 `.rdp` 文件后双击打开，输入用户名和密码即可连接。

### Q: 工单提交后多久会有回复？

工单的处理时间取决于分类设置的 SLA 时限和当前处理队列。紧急工单会优先处理。你可以在工单详情页查看 SLA 截止时间。

### Q: 注册时收不到验证码邮件？

1. 检查垃圾邮件文件夹。
2. 确认邮箱地址输入正确。
3. 部分邮箱后缀可能被系统限制，请联系管理员确认。
4. 如果持续收不到，请联系管理员。

### Q: 云电脑被禁用了怎么办？

请联系你的提供商或管理员了解原因，他们可以重新激活你的云电脑账户。
